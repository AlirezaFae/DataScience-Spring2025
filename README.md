# 📊 Personalized Notes & Codes - Data Science

> ⏳ **Work in Progress:** This repository is currently being updated and the notebooks will be added over time.

This repository is my forked and extended version of the DataScience-Spring2025 repository (Data Science Basics course at the University of Tehran, Spring semester). My goal in creating this repository is not merely to archive the materials, but to rewrite, enhance, and personalize the Python training notebooks to achieve a deeper understanding of the concepts in this field.

### 🌟 My Approach
In this project, I have utilized the Python training section of the course, using Professor Bahrak's teaching videos as my starting point. However, to reach my desired outcome, I have applied the following structural and content changes to the codes:

* **Renaming and restructuring files**
* **Adding foundational topics and tools:** I have added new sections to the tutorials.
